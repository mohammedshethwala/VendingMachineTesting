************************************************
CS 589 Project - Vending Machine - Instructions
************************************************

Steps to run the Test Driver STAProject.jar:
--------------------------------------------------------

1. From command prompt, go to the path of STAProject.jar
2. Run this command: java -jar STAProject.jar

or

1. Double click on the .bat file provided in the folder

--------------------------------------------------------
Steps to execute the test cases:

1.Enter an integer value from the options displayed and press 'Enter'.
2.If a method requires a parameter, you will be prompted to enter paramter's value.
3.Enter parameter value if any and press 'Enter'.
4.The return value will be displayed for the method which was called.

Extra Information:

1.The VendingMachine Class Methods are numbered from 1 to 10 
2.Testing Oriented Methods are numbered from 12 to 19
3.To quit, enter 11
